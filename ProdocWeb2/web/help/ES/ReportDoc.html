Informe Documentos OpenProdoc

Esta opción permite elegir un tipo de Informe y generarlo, con los datos de los documentos contenidos directamente (en el primer nivel) en la carpeta actual (siempre que el usuario tenga permiso de lectura sobre ellos).

Para ello debe situarse en la carpeta elegida y pulsar la onción de menú "Documentos -> Informe". Esta accion presenta un formulario con la lista de informes disponibles. Tras elegir el informe y pulsar Aceptar, se generará y descargará el informe elegido a partir de los documentos contenidos en la carpeta actual.
