Novedades de esta versión

Versión 2.1 (11/2017)

  • Punto de consulta (OPAC) parametrizable para permitir búsquedas de documentos y carpetas a usuarios puntuales/externos, obteniendo los resultados en formatos parametrizables.
  • Importación y exportación de documentos y carpetas/expedientes en modo Web.
  • Funciones de mover carpetas (y todo el árbol contenido) y documentos de un carpeta a otra.
  • Empaquetado de la versión portable en un solo paquete Web/Swing (desaparece la versión portable solo Swing).
  • Correción de algunos errores.
  • Correción de traducción.

Versión 2.0 (02/2017)

  • Interfaz de usuario Web totalmente rediseñada, incluyendo manejo de ventanas, listas ordenables, menús desplegables.
  • Administración y parametrización totalmente Web.
  • Nueva Versión Web multiusuario TOTALMENTE PORTABLE, no requiere instalación.

Versión 1.2 (11/2015)

  • Estándar RIS soportado para manejar referencias bibliográficas a documentos.
  • Creación de informes personalizados y formatos de exportación (html, XML, csv, txt, RIS)
  • Búsqueda por texto completo de los documentos.

Versión 1.0 (9/2014)

  • Soporte de Tareas para el mantenimiento y ciclo de vida de los documentos y carpetas.
    • Tareas programadas.
    • Tareas asociadas a eventos
      • Integradas con el evento
      • Diferidas
  • Arquitectura Extendida, incluyendo "Bridge Servers" para ofrecer mayor seguridad e instalación mas sencilla
  • Almacenamiento de documentos en Amazon AWS S3 (y otros sistemas compatibles con Amazon S3).

Versión 0.8 (/7/2013): Versión RC1

  • Soporte de Tesauros
    • Gestión completa de múltiples Tesauros.
    • Importación y Exportación en formato estándar SKOS.
    • Validación de metadatos de tipos documentales y carpetas contra valores de los tesauros.
  • Modificación de Tipos de documento y carpeta ya en uso, con documentos introducidos.
  • Importación de documentos digitalizados y clasificados desde Kofax Capture y Abbyy Finereader

Versión 0.7 (/5/2012): Versión Beta pública

  • Distribución Portable multiplataforma (Linux, Windows, Mac)
  • Atributos Multivaluados en nucleo y clientes
  • Repositorios de documentos referenciados Internet
  • Administración Mejorada
  • Corrección algunos errores potenciales (Atributos, manejo arbol swing, Web)

Versión 0.6.1 (29/12/2011): Versión Beta para recogida de opiniones

  • Mejoras comportamiento arbol Swing
  • Carpetas base ACL público por defecto
  • Corrección error versionado Swing y Web

Versión 0.6 (6/12/2011): Versión Beta para recogida de opiniones

  • Documentación en Inglés
  • Importación y Exportación de Archivos y Carpetas
  • Correciones soporte idiomas y documentación en Install y Setup
  • Reducción tamaño máximo atributos a 254 (problemas Access y ¿....?)
  • Correción ubicación tras ordenar tablas en Swing
  • Actualización ACL/permisos usuario actual tras crear usuario
  • Obligatoriedad de nombres visibles y descripciones a todos los niveles

Versión 0.5 (1/10/2011): Versión Beta para recogida de opiniones

  • Nucleo básico
  • Cliente Swing de administración y acceso
  • Cliente Web sin administración
  • Soporte metadatos en BBDD
  • Repositorios en FileSystem, BBDD/BLOB y ftp
  • Autenticación OPD, LDAP, BBDD, SO
