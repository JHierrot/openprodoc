What is new in this version of OpenProdoc

Version 3.0 (11/2019)

  • New API REST for better integration.
  • Posibility of developing extensions personalized for:
    • Event Tasks
    • Scheduled Tasks
    • Repository Drivers for storing the documents
    • Users Authenticators
  • Improved security:
    • Portable version includes https (you must include your certificates and review keystore)
    • APIs REST and Remote can be disabled
    • jwt tokens supported
  • New toolbar in web application.
  • Support for SQL.
  • Improvements in Sessions support.
  • Some bugs fixed.

Version 2.3 (4/2019)

  • Functions to define contribution systems so that people outside the organization or department can provide and exchange documents and have their own files without having to register.
  • Possibility of generating reports on current folders and documents.
  • The full-text indexing language of the documents can be defined, as well as the dictionary of stopwords, so that searches are optimized.
  • Possibility of creating "packages" or projects containing definitions of all types of elements, thesauri and folder and document structures.
  • Greater control of sessions and resources as well as usage monitoring and connected users.
  • Minor improvements and bug fixes.

Version 2.2 (9/2018)

  • Version 2.2 has been completely recompiled to be able to work with a MINIMUM version of Java 1.8 and higher versions.
  • In addition, the code includes several improvements and better error control.

Versión 2.1 (11/2017)

  • Configurable Search Forms (OPAC) allowing external users to search documents and folders, obtaining results in configurable formats.
  • Import and export of documents and folders in web interface.
  • Functions for moving folders (and alll the contained tree) and documents from one folder to another.
  • Unified package of the portable distribution, containing both Web and Swing versions (the portable Swing only distribution disappears).
  • Some errors fix.
  • Translation fix.

Versión 2.0 (02/2017)

  • New Web interface completely redesigned, including manage of windows, advanced grids and menus.
  • Administration and management Web.
  • New Multi-user Web Version TOTALLY PORTABLE, no installation required.

Version 1.2 (11/2015)

  • RIS standard format support for document references.
  • Creation of personalized reports and export formats (html, XML, csv, txt)
  • Documents and metadata Full Text search.

Version 1.0 (9/2014)

  • Tasks support for maintenance and lifecycle of documents and folders
    • Programed Task.
    • Event Tasks
      • Transactional
      • Deferred
  • Architecture Extended, now supporting "Bridge Servers" for more security and easier installation
  • Storage of documents in Amazon AWS S3 (and other compatible S3 Storage Systems).

Version 0.8 (/7/2013): Version RC1

  • Thesaurus Support
    • complete Thesaurus management.
    • Skos-rdf standard support for import-export.
    • Multiple thesaurus.
    • Validation of selected metadata against thesaurus.
  • Modification of definitions already in use.
  • Import of document scanned and clasified in Kofax Capture or Abbyy Flexicapture

Version 0.7 (5/2012): Public Beta Version

  • Portable distribution Multiplatform (Linuax, Windows, Mac)
  • Multivalued Attributes in engine and clients
  • Internet referenced repositories of documents
  • Enhanced Administration
  • Correction of some potential problems (Attributes, Swing tree management, Web)
