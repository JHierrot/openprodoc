CSS OPAC

For customization of the OPAC of OpenProdoc in the style of the institution that is using it, style sheets can be used that will be referenced in the configuration file. A CSS can be referenced by several files of configuration of OPAC.

The styles defined for in the html of the Query Point are:

  • OPACBODY : html body.
  • OPACFORM : Form.
  • OPACTABLE : Table that containf the fields of the form.
  • OPACFS : Style of the FieldSet.
  • OPACLEG : Legend.
  • OPACDT : Label of the combo of document types.
  • OPACCOMB : Combo of document types.
  • TD_OPACCOMB : Tooltip help of document types.
  • OPACFTLAB : Label of the full text search field.
  • OPACFTINP : Full text search field.
  • TD_OPACFTINP : Tooltip help of the full text search field.
  • OPACFORMATLAB : Label of the ouput reports combo.
  • OPACFORMATCOMB : Ouput reports combo.
  • TD_PACFORMATCOMB : Tooltip help of the ouput reports combo.
  • OPACLAB : Labels of the input fields.
  • OPACINP : input fields.
  • TD_OPACINP : Tooltip help of the input fields.
  • OPACFORMATTHES : Thesaurus input fields.
  • OPACBUT : Search button.
  • OPACHELP : Help button.

Of course you can redefine any style of the standard html elements (<h3>,<img>, ...)
