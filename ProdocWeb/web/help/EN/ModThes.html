Modify Thesaurus OpenProdoc

This form allows modify an existing Thesaurus, filling its data. The Thesaurus id can't be changed.

You can fill the fields:

  • PDID: Numeric identifier of the thesaurus. A number between 1 and 999999. It will be used as a reference when defining metadata validated against values in a specific thesaurus
  • Name: Name of the thesaurus (up to 128 characters).
  • Definition: Description of the thesaurus (up to 254 characters)
  • Language: 2 characters ISO of the language (ES, EN, PT, etc)
  • Note: Scope Note of the thesaurus (SCN) (up to 254 characters).

In the event of an error (lack of user permissions, communication error, incorrect date, etc.), the operation is canceled and the system will communicate the reason for the error.
