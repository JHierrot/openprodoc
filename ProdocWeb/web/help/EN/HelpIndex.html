Help Index OpenProdoc

  • Introduction: General introduction to the characteristics of OpenProdoc.
  • Main Window: Shows an overview of the main window.
  • Folders
    • Add Folder: Adds a folder introducing only the name.
    • Delete Folder: Deletes the current folder, asking confirmation for the operation.
    • Update Folder: Modifies the current folder.
    • Add Folder in extended mode: Adds a folder selecting type and introducing all the metadata.
    • Update Folder in extended mode: Modifies any metadata of the current folder.
    • Search Folder: Search of folders by different criteria and metadata.
  • Documents
    • Add Document: Adds a document, introducing a minimum of information.
    • Delete Document: Deletes the current document, asking confirmation for the operation.
    • Add Document in extended mode: Adds a document selecting type and introducing all the metadata.
    • Update Document in extended mode: Modifies any metadata of the current document.
    • CheckOut Document: Locks the current document for editing.
    • CheckIn Document: Confirms changes in a locked document.
    • Cancel Checkout of Document: Cancel changes in the document.
    • List Versions of Document: Lists all versions of the current document.
    • Search Documents: Search of documents by different criteria and metadata.
  • Other Tasks
    • Paper Bin: Shows the deleted documents de every type, allowing to recover them or to delete them permanently.
    • Change Password: Allows to change the password (if the kind of authentication allows the operation ).
  • Administration (This options will be visible or not depending on version of OpenProdoc client and the user permissions)
    • ACL: Allows to manage the permissions (Access Control List) used in all the elements of OPD.
      • ACL maintenance: Details of the ACLs and its edition.
    • Groups: Allows to manage the groups over which the user has permissions.
      • Groups maintenance: Details of the Groups and its edition.
    • Users: Allows to manage the Users.
      • Users maintenance: Details of the Users and its edition.
    • Roles: Allows to manage the users' roles, setting the authorized tasks.
      • Roles maintenance: Details of the Roles and its edition.
    • Mime Types: Allows to manage the mime types and the linked extensions.
      • Mime Types maintenance: Details of the Mime Types and its edition.
    • Repositories: Allows to manage the repositories in which store the documents.
      • Repositories maintenance: Details of the Repositories and its edition.
    • Objects definitions: Allows to manage the documents types and the folders types (containers).
      • Objects definitions maintenance: Details of the Objects definitions and its edition.
      • Metadata maintenance: Metadata definition for the document types.
    • Authentication Systems: Allows to manage the different kind and origins of users authentication.
      • Authentication Systems maintenance: Details of the Authentication Systems and its edition.
    • Customizations: Allows to manage the Customizations (styles, language, etc).
      • Customizations maintenance: Details of the Customizations and its edition.
  • Other information
    • Install: Instructions for installations and setup.
    • Users interfaces: Description of the general criteria for users interfaces and differences between client version (Web and Swing).
    • Languages support: Reference of the supported languages and the localization criteria in current version.
    • Reporting bugs and problems: How to detect and report problems and bugs in OPD.
    • What Is New: Contains descriptions of what changed in this version and previous versions.
    • About..: References to authors and licences.
