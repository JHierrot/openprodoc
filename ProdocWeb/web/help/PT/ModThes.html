Modificar Tesauro OpenProdoc

Este formulario permite modificar metadatos del Tesauro seleccionado. No permite modificar el identificador de tesauro.

Puede introducirse la información siguiente:

  • Nombre: Nombre del tesauro (hasta 128 caracteres).
  • Definición: Texto descriptivo del tesauro (hasta 254 caracteres)
  • Idioma: 2 caracteres ISO del idioma (ES, EN, PT, etc)
  • Nota: Nota del tesauro (SCN) (hasta 254 caracteres).

En caso de producirse un error (carencia de permisos del usuario, error de comunicaciones, etc), la operación se cancela y se comunicará al usuario el motivo del error.
