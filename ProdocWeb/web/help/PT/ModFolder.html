Modificar Carpeta OpenProdoc

Este formulario permite modificar el nombre de la carpeta seleccionada. No permite modificar otros metadatos o la seguridad, para lo que debe utilizarse la modificación extendida.

Solo debe introducirse la información siguiente:

  • Título: Nombre descriptivo de la carpeta

Los campos obligatorios para cada tipo apareceran resaltados. Al situarse el cursor o ratón sobre cada campo, aparecerá un mensaje emergente o un texto en la banda de inferior del formulario con información sobre el campo.

En caso de producirse un error (carencia de permisos del usuario, error de comunicaciones, etc), la operación se cancela y se comunicará al usuario el motivo del error.
