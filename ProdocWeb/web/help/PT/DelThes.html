Borrar Tesauro OpenProdoc

Al seleccionar el borrado de un Tesauro se presentará esta pantalla que muestra toda la información deL mismo, de forma que pueda confirmarse antes de borrar que es el elemento correcto.

Si se acepta el borrado, el tesauro y TODOS los términos incluidas se borran definitivamente.

Si los términos del tesauro son utilizados como valores para la entrada de un metadato de un tipo documental o tipo de carpeta, no podrá borrarse el tesauro.
