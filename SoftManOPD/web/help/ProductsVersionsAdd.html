Help SMO Product Version Add

This form allows to add a new version to a Product. All the fields follow the interface conventions of OpenProdoc:

  • Standard fields: Without special mark
  • Required fields: With a character "*"
  • Thesaurus controlled: With a button with character "T" for selecting and "-" for deleting
  • Multivalued: With a button with character "*"
  • Multivalued and Thesaurus controlled: With a button with characters "T*"

With the default configuration, the fields to fill are:

  • Tille-Name: Name of the version
  • License: Type of License (Comercial, Open Source, GPL, Affero, BSD,..) of the version
  • Date Initial: Date when the productversion was (will be) initialy available
  • Date End Support: Date when the STANDARD support of the product version will end
  • Date End Support Extended: Date when the EXTENDED support (if defined) of the product version will end
  • Notes: Some notes about the version

In SMO Tree there is a complete view of operations and forms.
