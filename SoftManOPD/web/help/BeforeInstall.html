Help SMO Before Install

The Software Management for OpenProdoc (SMO) requires OpenProdoc 3.0.3 or later.

Besides that, SMO will work in any platform and infrastructure supported by OpenProdoc.

The page Install details the installation process over an OpenProdoc installation.

If you don't have OpenProdoc already installed, please install it before installing SMO. If you plan to use SMO in a portable environment, there is a SMO portable version available for downloading from the OpenProdoc site, ready for downloading, unziping and start, without any additional operation.

Main Win Help contains a global description of the this extension for OpenProdoc.

In SMO Tree there is a complete view of operations and forms.
