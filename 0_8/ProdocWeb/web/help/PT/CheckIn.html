CheckIn Documento OpenProdoc

La opción de CheckIn CONFIRMA todos los cambios que se han realizado sobre un documento y elimina las copias de trabajo del usuario en el servidor, publicando como versión vigente la nueva versión.

Debe introducirse un código o nombre de versión significativo que permita identificarlo y recuperar la versión cuando se consulte la lista de versiones del documento

Ver: CheckOut (Confirmar cambios en Documento Bloqueado) y Cancel Checkout (Cancelar Cambios en Documento)
