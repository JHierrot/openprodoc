Mantenimiento Personalizaciones

Al seleccionar en el formulario de Lista Personalizaciones las opciones de Alta, Baja, Modificación o Copia de Personalizaciones aparecerá este formulario con diversos campos activados o desactivados.

Este formulario contiene la información:

  • Nombre identificativo: Identificador de la Personalización cuyo valor no puede repetirse ni cambiar una vez asignado. Puede tener una longitud máxima de 32 caracteres. (Ej. "RRHH_Portugal")
  • Descripción de la Personalización: Descripción de la Personalización que permita interpretar y recordar el sentido de la misma. Puede tener una longitud máxima de 128 caracteres. (Ej.: "Personalización en Portugues para Recursos Humanos Portugal")
  • Idioma del interfaz de usuario: Idioma en que se presentará todos los mensaje y literales de OPD. Los literales de los elementos (Documentos, Carpetas, etc) se presentarán siempre tal como fueron creados, en esta versión de OPD no hay soporte para objetos multiidioma. Se utilizan los códigos ISO 639-1 de dos caracteres. Para ver los detalles de idiomas consultar Idiomas
  • Formato de entrada y presentación de las Fechas: Formato en que se presentará la fecha. (Ej. "MM/dd/yyyy")
  • Formato de entrada y presentación de las Fecha y hora: Formato en que se presentará la fecha y hora . (Ej. "MM/dd/yyyy HH:mm")
  • Path de parametrización para el cliente Web: Path relativo dentro de la aplicación web donde se encuentran todos los elementos propios de la apariencia. OPD Incluye algunos estilos en diversos subdirectorios que pueden copiarse, modificando únicamente los elementos deseados. Cuando se indica un path debe crearse siempre TODOS los elementos necesarios (CSS, imágenes, Javascript, etc) por lo que lo mejor es copiar y modificar los elementos deseados.
  • Fuentes a utilizar en el cliente Java Swing: Fuentes a utilizar para el interfaz Swing. Para cada elemento se indica el tipo de fuente, atributos de la fuente (según las definiciones de awt.font) y el tamaño para los elementos:
    • Menú
    • Arbol de carpetas
    • Listados de elementos
    • Cuadros de diálogo
