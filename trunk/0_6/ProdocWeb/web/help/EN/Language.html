Language Support

OPD is developed with the idea of it being easy to translate to another language, so that it can meet the needs of users. Localization or adaptation to a language is structured in 3 parts:

  • Core and Clients (Web and Swing)
  • User Documentation
  • Administration Documentation

In this version, the localization covers:

LanguageCore and Clients User Doc.Administration Doc.
    SpanishYes Yes Yes
    EnglishYes Yes Yes
    PortugueseYes No No

Regarding location should be noted that the date format is covered with customization ( Custom List ), so that support for countries sharing a language, but a different format date is just translation.
