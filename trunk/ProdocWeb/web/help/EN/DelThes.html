Delete Thesaurus OpenProdoc

When select the option "Thesaurus->delete" it will be displayed this form showing all the information related to the thesaurus; this allows to review that is the correct thesaurus to be deleted.

If you accept to delete the Thesaurus, the Thesaurus itself and ALL the contained terms will be deleted.

If any of the terms to be deleted is used (referenced) in a document or folder, the deletion will be anulated.
