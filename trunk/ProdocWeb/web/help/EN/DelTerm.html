Delete Term OpenProdoc

When select the option "Terms->delete" it will be displayed this form showing all the information related to the term; this allows to review that is the correct term to be deleted.

If you accept to delete the term, the term itself and ALL the specific (contained) terms will be deleted.

If any of the terms to be deletd is used (referenced) in a document or folder, the deletion will be anulated.
