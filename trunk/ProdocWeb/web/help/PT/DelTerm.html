Borrar Término OpenProdoc

Al seleccionar el borrado de un Término se presentará esta pantalla que muestra toda la información deL mismo, de forma que pueda confirmarse antes de borrar que es el elemento correcto.

Si se acepta el borrado, el término y TODOS los términos específicos a todos los niveles incluidas se borran definitivamente.

Si los términos son utilizados como valores para la entrada de un metadato de un tipo documental o tipo de carpeta, no podrá borrarse el término.
